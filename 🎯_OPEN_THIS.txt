  📱 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ADMIN WORKFLOW:
1. Login as Admin
2. Add teachers with IDs/passwords
3. Upload time tables
4. Monitor all attendance
5. Generate reports
6. Configure system settings

TEACHER WORKFLOW:
1. Login as Teacher
2. Register students:
   → Upload clear face photo
   → Fill all details (name, roll, semester, class, phones)
   → Generate login credentials
   → SMS sent to student & parent
3. Mark attendance:
   → Start camera
   → Students scan faces
   → System verifies distance & liveness
   → Auto-marks if all checks pass
4. Verify attendance:
   → Review all entries
   → Approve or reject
   → Flag proxy attempts
5. View class reports

STUDENT WORKFLOW:
1. Login with credentials (received via SMS)
2. Mark attendance:
   → Go to Mark Attendance page
   → Start camera
   → Position face clearly
   → System checks:
      - Distance (≤20m from teacher)
      - Face detection
      - Liveness (real vs photo)
   → If all pass → Marked ✓
   → SMS sent to parent
3. View attendance history
4. Check time table
5. Read notifications
