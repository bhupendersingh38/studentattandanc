🔧 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════

Problem: "Backend is not running" keeps showing
Solution:
  1. Check the backend terminal is open
  2. Look for "Uvicorn running" message
  3. Wait 5 seconds after opening
  4. Refresh browser (F5)

Problem: Can't see backend terminal
Solution:
  1. Look at your taskbar for Command Prompt windows
  2. Or just run START_BACKEND_ONLY.bat again
  3. You'll see a black window with text

Problem: Nothing happens when I login
Solution:
  1. Open browser console (F12)
  2. Check for errors
  3. Make sure backend is running
  4. Try refreshing
