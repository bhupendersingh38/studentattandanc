🔧 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════

    Problem: Backend won't start
    Solution: cd backend && python main_production.py

    Problem: Frontend shows errors
    Solution: cd frontend && npm install

    Problem: Port already in use
    Solution: netstat -ano | findstr :8000
             taskkill /PID <PID> /F

    Need help? Run: CHECK_SYSTEM.bat
