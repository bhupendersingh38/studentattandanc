🔧 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════

Problem: "Backend is NOT responding"
Solution: 
  1. Refresh browser (F5)
  2. Wait 2-3 seconds
  3. Should connect automatically

Problem: Login doesn't work
Solution:
  1. Check credentials are correct
  2. Press F12 to open console
  3. Check for errors
  4. Refresh and try again

Problem: Blank page
Solution:
  1. Make sure you're using this URL:
     http://localhost:3000/app.html
  2. Not opening the file directly
  3. Refresh the page
